# One Hot Encoding for Categorical Data

## 📌 Project Overview
This project demonstrates how to convert categorical data into numerical format using One-Hot Encoding. A categorical column is encoded into binary values such as "Yes" and "No" to make the dataset suitable for machine learning models.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Identifies categorical columns in the dataset
- Applies One-Hot Encoding
- Converts categorical values into binary (Yes/No) representation
- Prepares data for machine learning models

## 🔧 Method Used
One-Hot Encoding:
- Categorical values are transformed into binary columns.
- Each category is represented by a new column.
- Values are encoded as Yes/No (or 1/0) depending on the category presence.

## ⚙️ How to Run the Project
1. Clone or download the repository.
2. Open the Jupyter Notebook file.
3. Run the notebook cells step by step.
4. The notebook will demonstrate One-Hot Encoding on the dataset.

## 🎯 Purpose
Machine learning models require numerical data. One-Hot Encoding is an important preprocessing technique used to convert categorical data into a format that algorithms can understand.
